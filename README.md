# TerraBloom

**TerraBloom** is a modern, interactive React landing page for a botanical collection. It combines **TailwindCSS** for styling and **Framer Motion** for smooth animations to create a visually appealing user experience.

---

## Features

- Smooth scrolling animations and fade-in effects using `framer-motion`.
- Hero section with animated text and image carousel.
- Sections for About, Collection, Flower Info, and Testimonials.
- Fully responsive design for mobile, tablet, and desktop.
- Clean and modular React component structure.
- TailwindCSS for utility-first styling.

---

## Technologies Used

- **React 18** – Frontend library
- **Vite** – Fast development and build tool
- **TailwindCSS** – Utility-first CSS framework
- **Framer Motion** – Animations and transitions
- **TypeScript** – Static typing
- **PNPM** – Fast package manager

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/terra-bloom.git
cd terra-bloom
```

### 2. Install dependencies
```bash
pnpm install
```

### 3. Start the development server
```bash
pnpm dev
```

### 4. Build for production
```bash
pnpm build
```
